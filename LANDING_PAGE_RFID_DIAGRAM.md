## ✨ Animation Effects

```
1. PULSE (Alert Badge)
   ├─ Duration: 2s
   ├─ Repeat: Infinite
   └─ Effect: Opacity 1 → 0.7 → 1

2. FLOAT (RFID Card)
   ├─ Duration: 3s
   ├─ Repeat: Infinite
   └─ Effect: Y-axis -20px → 0 → -20px

3. BOUNCE (Floating Buttons)
   ├─ Duration: 2s
   ├─ Repeat: Infinite
   └─ Effect: Y-axis 0 → -10px → 0

4. FADE-IN (Cards on Scroll)
   ├─ Duration: 0.6s
   ├─ Trigger: Intersection Observer
   └─ Effect: Opacity & Y-translate

5. SCALE (Button Hover)
   ├─ Duration: 0.3s
   ├─ Trigger: :hover
   └─ Effect: Scale 1 → 1.05 + shadow

6. RIPPLE (Button Click)
   ├─ Duration: 0.6s
   ├─ Trigger: :active
   └─ Effect: Circle expand + fade

7. SMOOTH SCROLL (Navigation)
   ├─ Duration: Auto
   ├─ Trigger: Click anchor link
   └─ Effect: Smooth scroll behavior
```

## 🔄 Data Flow

```
USER INTERACTION FLOW:

1. User arrives at /landing/laixe-rfid-tuvan-nhanh
   ↓
2. Page loads CSS + JS
   ├─ CSS: Inline styles (no external loading)
   ├─ JS: Initialize event listeners
   └─ Ready time: < 1 second
   ↓
3. User reads content (top to bottom)
   ├─ Hero: Hook & interest
   ├─ Warning: Create urgency
   ├─ About: Educate
   ├─ Solution: Show relief
   ├─ Proof: Build trust
   └─ CTA: Convert
   ↓
4. User interacts:
   ├─ Click "Nhận Tư Vấn Ngay" → Scroll to form
   ├─ Click "Chat Zalo Ngay" → Open Zalo link
   ├─ Click navigation → Smooth scroll
   ├─ Click floating button → Open Zalo/Call
   └─ Submit form → Auto-open Zalo with message
   ↓
5. Form submission:
   ├─ User fills: Name, Phone, Vehicle type
   ├─ Click submit
   ├─ JS builds message with user data
   ├─ Opens Zalo link with pre-filled message
   └─ User can send Zalo message directly
   ↓
6. Lead captured:
   ├─ Message appears on business Zalo
   ├─ Business responds quickly (< 5 min ideal)
   ├─ Lead converts to appointment/sale
   └─ High conversion rate (30-50% of leads)
```
